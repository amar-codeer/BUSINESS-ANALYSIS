# Indian Business Category Analysis Dashboard

## Project Overview
This project analyzes Indian business category data to identify trends, patterns, and business insights using Python and Power BI.

## Objectives
- Clean and preprocess raw business data
- Perform exploratory data analysis (EDA)
- Identify key business trends
- Build an interactive Power BI dashboard

## Tools & Technologies
- Python
- Pandas
- NumPy
- Matplotlib
- Power BI

## Project Workflow
1. Data Collection
2. Data Cleaning
3. Data Transformation
4. Exploratory Data Analysis
5. Dashboard Development
6. Business Insights Generation

## Dashboard Features
- Category-wise Analysis
- Revenue & Performance Tracking
- Interactive Filters
- KPI Monitoring
- Trend Analysis
